# BreastCancerCNN

A modular convolutional neural network implementation for classifying breast cancer histology images. This project includes custom implementations of neural network layers, data loading utilities, model assembly, training pipeline, and plotting tools.

---

## üß† Features

- Custom layers:
  - `activation.py`: Activation functions like ReLU, sigmoid, etc.
  - `conv.py`: Custom convolutional layer implementation.
  - `dense.py`: Fully connected (dense) layer.
- Modular model architecture defined in `model.py`
- Data loading handled in `loading.py`
- Training and evaluation entry point in `main.py`
- Plotting and visualization tools in `plotting.py`
